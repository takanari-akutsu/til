# 演算子とは
- 計算するための記号


# 代入演算子（.）
- 左右の値を結合できる記号
- 変数に代入する場合にも用いる


# 四則演算
- 加算（+）
- 減算（-）
- 乗算（*）
- 除算（/）
- 剰余（%）:余りを求める際に使用する。奇数ごとの処理や〇倍の時の処理をしたいときに用いる。
- （）を用いることで、計算順序を優先することができる


# 論理演算子
- 論理積（&&）:AND
- 論理和（||）:OR
- 否定（!）:NOT


# 比較演算子
- 「==」:等しい
- 「!=」:等しくない
- 「<」:少ない
- 「>」:大きい
- 「<=」:以下
- 「>=」:以上
